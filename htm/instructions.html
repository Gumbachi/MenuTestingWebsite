Instructions

Intro

This experiment involves testing different menus. You will be given 15 tasks for each menu. The task will require you to find a menu item. You will be timed for each task.

Starting

To start the experiment press the start button in the dialog box present in the middle of the screen. It will look like this.


The dialog box also shows the current task.

When you are done with looking at the instructions, click the Start Experiment button to move to the first menu.
